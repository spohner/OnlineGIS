This page describes all functions "Gisse GIS" is capable of. If you need further help please contact me. Contact information can be found here. The code for "Gisse GIS" is freely available at Github.

Buffer

Extends selected area with a given range in meters. Useful for proximity analysis.

Area

Returns the selected layer' area in squaremeters.

Merge

Takes two layers and combines them into one, containing the features of both.

Subtract

You select two layers. One base layer and a layer to subtract. Returns the subtracted layer.

Intersect

Returns the overlap of two layers.

Distance

Gives the shortest distance in meters between two layers.

Simplify

Simplifies the shape of a layer using the Douglas-Peuker algorithm.
